# Back-end Razvoj

Tukaj se nahaja strežniška logika, API-ji in povezave z bazo.

### Navodila:
- Dokumentirajte svoje API točke (endpoints), da bo Front-end ekipa vedela, kako jih uporabiti.
- Ne nalagajte datotek z gesli ali osebnimi ključi (uporabite `.env` datoteke, ki so v `.gitignore`).
- Redno preverjajte skladnost z bazo podatkov.
