# Razvoj kode (Development)

Tukaj se nahaja vsa izvorna koda projekta.

### Navodila:
- Preden začnete pisati kodo, preverite, ali so vse knjižnice navedene v `.gitignore`.
- Vsaka večja sprememba naj ima svojo vejo (branch).
- **Cybersecurity:** Prosimo, da tukaj dokumentirate morebitne varnostne popravke.
