# ProkrastinatorGPT

ProkrastinatorGPT je razširitev za brskalnik (Browser Extension), zasnovana za študente, ki želijo optimizirati svoj čas pri delu z digitalnimi učnimi gradivi. Orodje s pomočjo umetne inteligence (LLM) analizira kompleksna navodila nalog v spletnih učilnicah (Moodle) in študentu ponudi ključne povzetke, oceno zahtevnosti ter predviden časovni okvir za uspešno oddajo.

## 🚀 Ključne funkcije

- **AI Povzetek navodil:** Samodejna ekstrakcija bistvenih zahtev iz dolgih besedil navodil.
- **Načrt reševanja po korakih:** Generiranje logičnega zaporedja opravil za dosego cilja.
- **Predikcija časovnega okvirja:** Ocena potrebnega časa za reševanje na podlagi kompleksnosti naloge.
- **Analiza težavnosti:** Kategorizacija nalog glede na zahtevane veščine in pretekle podatke.
- **Prilagodljiva obvestila:** Diskretni popup elementi, ki se aktivirajo ob zaznavi novih učnih vsebin.

## 🛠 Tehnologije

- **Frontend:** JavaScript (ES6+), HTML5, CSS3 (Custom Components)
- **Backend/AI:** OpenAI API (GPT-4o) ali alternativni LLM modeli
- **Platforma:** Web Extensions API (združljivo s Chrome, Edge, Firefox)
- **Varnost:** Lokalna hramba API ključev in anonimizacija podatkov iz učilnice.
