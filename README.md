# 💤 ProkrastinatorGPT

### *Ker začeti pravočasno ni osebnostna lastnost*

---

**ProkrastinatorGPT** je browser extension za študente, ki:

* želijo porabiti čim manj časa za faks
* radi po 3 ure gledajo Reelse
* nočejo brati navodil


💡 S pomočjo AI:

* prebere navodila namesto tebe
* ti pove, koliko časa *dejansko še lahko nič ne delaš*
* in te zaščiti pred overachiever prijatelji

Orodje samodejno prebere navodila iz Moodle-a in ti pove samo to, kar te dejansko zanima:

koliko moraš narediti in kako dolgo lahko še ignoriraš vse skupaj.
---

## 🎯 Poslanstvo

> “Zakaj bi začel danes, če lahko začneš jutri?”

### ✅ Maksimiziramo

* doomscrollanje (Instagram, TikTok, YouTube Shorts… vse šteje)
* “bom kasneje” mindset
* čas za binge watchanje TV serij
* čas za druge stvari (derankanje v lol-u)

### ❌ Minimiziramo

* stres
* dejansko delo
* potrato časa

---

## 🚀 Ključne funkcije

### 🧠 AI Povzetek navodil

Prebere 3 strani Moodle teksta in ti pove:

> “Nared PowerPoint pa nekaj napiš zraven.”

Potem lahko njegov povzetek preprosto kopiraš v drug LLM in ti naredi nalogo
---

### 🪜 Načrt reševanja po korakih

Strukturiran plan, ki ga boš ignoriral in prilepil v LLM

---

### ⏳ Napoved časa reševanja

Najbolj iskrena funkcija v projektu:

koliko časa dejansko potrebuješ

---

### 📊 Analiza težavnosti

* 🟢 "Bo chat naredu v 1 promptu"
* 🟡 "Claude will cook"
* 🔴 "Bo treba PRO paket kupit"

## 🛠 Tehnologije

Dejansko smo nekaj naredili:

* **Frontend:** React + TypeScript + Vite (HTML5, CSS3)
* **Backend:** Node.js + Express 5
* **AI integracija:** OpenAI Node SDK (z možnostjo alternativnih LLM)
* **Podatkovna plast:** PostgreSQL (`pg`, `node-pg-migrate`) + MongoDB (`mongoose`)
* **Platforma:** Browser Extension (Chrome/Edge/Firefox) + Web Extensions API
